# Content Rights — Quiz Pszczelarski

> **Location**: App Store Connect → App Information → Content Rights  
> **Question**: Does your app contain, show, or access third-party content?

---

## Correct Answer

```
○ Yes, it contains, show, or accesses third-party content, 
  and I have the necessary rights

◉ No, it does not contain, show, or access third-party content  ← SELECT THIS
```

---

## Why "NO"?

All quiz content in Quiz Pszczelarski is **proprietary**:

| Content Type | Source | Third-Party? |
|--------------|--------|--------------|
| Quiz questions | Proprietary (Rafał) | ❌ NO |
| Quiz answers & explanations | Proprietary | ❌ NO |
| Animations (Lottie) | Licensed open-source | ✅ Licensed |
| Icons/Graphics | App assets (Material Design) | ✅ Open license |
| Firebase services | Google (backend) | ⚠️ Not "content" |
| Music/Sounds | Custom or royalty-free | ✅ Licensed/Custom |

**Key distinction:**
- "Third-party content" = embedded media (videos, images, text, articles, quizzes from other sources)
- "Services" = Firebase, Analytics (these are tools, not content being displayed)

---

## What Counts as Third-Party Content?

### ❌ Would be "YES" if:
- Quiz questions copied from textbooks
- Images from Getty Images / Unsplash
- Music from a band (without license)
- Articles/blog posts embedded
- Video tutorials from YouTube
- Fonts from external sources (without license)

### ✅ Our situation: "NO" because:
- All quiz questions = proprietary created
- All animations = self-made or properly licensed
- All graphics = app assets (Material Design)
- No embedded media from external sources
- No third-party educational content

---

## If You Had Third-Party Content

If selecting "YES", you would need to:
1. List all third-party content sources
2. Provide proof of licensing/rights
3. Submit copyright documentation
4. May require contractual agreements

**Example:**
- Quiz questions from "Beekeeping Manual 2024" → Need license from publisher
- Images from photographer → Need attribution/license
- Sound effects from freesound.org → Need to verify license (some require attribution)
